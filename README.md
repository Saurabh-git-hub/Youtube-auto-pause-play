# YouTube Auto-Pause Extension

Have you ever been watching a YouTube video, switched tabs for a second… and then realized you missed a part of it?  
Well, I’ve built a Chrome Extension that solves exactly this problem! 🎉

## ✨ What it does

- **Automatically pauses** your YouTube video when you leave the tab
- **Automatically resumes** the video when you come back
- Works smoothly without disturbing your browsing experience

## ✨ Why this is useful?

- ✔️ **No more missing important parts** of a video
- ✔️ Perfect for **multitaskers** who often switch tabs
- ✔️ Saves you from the hassle of manually pausing/playing every time

## ⚡ How to use it?

1. Download the repo from GitHub
2. Open web-browser and type web-browser-name://extensions/ → example  `chrome://extensions/`
3. Turn on **Developer Mode** (top right)
4. Click **Load unpacked** → Select the downloaded folder
5. Done ✅ (Now enjoy distraction-free YouTube!)
